Formless Forms
Formless Forms are a small Angular components library with the objective of providing components that can be combined in any fashion to create dynamic forms with validation that do not require html template forms. Formless Forms are flexible, dynamic, and extendable across frameworks / applications.
Please contribute anytime by submitting issues, pull requests, and/or by using this small components library in your projects. We would love to hear your feedback and welcome community contributions.
Why Formless
Simplicity is the key when it comes to formless form components. Any developer that has spent a short amount of time in Angular should be able to review the source code and come away with a clear understanding of how the components work.
You have our full permission to review and duplicate any and all code . With little effort you should be able to do a lot of customization for your forms using these components.
The overall objective of formless-forms has always been to simplify components, forms, and minimize code.
Getting Started: 2 Step Installation
Step 1: npm install formless-forms package
Step 2: import component module(s)
